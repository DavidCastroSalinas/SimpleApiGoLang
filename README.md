# SimpleApiGoLang
Construcción de una API REST en base a lenguaje GO Lang (Ejemplo de bautismos)

## Obtener librería Gorilla/MOX
''''console
go get -u  github.com/gorilla/mux
''''

##Luego se deberá ejecutar
''''console
go run main.go 
''''
